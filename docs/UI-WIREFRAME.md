# 小学数学每日练习小程序 — UI 线框图与交互说明

## 设计规范

| 维度 | 规范 |
|------|------|
| **主色调** | 高饱和蓝 `#2E6FF2` / `#3478F6`（页面顶部背景 + 导航栏 + 按钮） |
| **正确反馈色** | 绿色 `#66BB6A` |
| **错误反馈色** | 红色 `#EF5350` |
| **提示/漏选色** | 橙色 `#FFA726` |
| **内容背景色** | 白色 `#FFFFFF`，带大圆角从蓝色背景过渡 |
| **卡片背景色** | 浅蓝 `#EBF2FF`（题目区域 / 信息卡片） |
| **正文字号** | 题目 20-24pt，按钮 18pt，算式 36-48pt |
| **圆角** | 页面顶部 40rpx，卡片 24rpx，按钮 16rpx |
| **最小点击区域** | 44×44pt |
| **动画时长** | 反馈动画 0.3s，页面切换 0.25s |
| **字体** | 系统默认字体，数字用等宽字体 |
| **页面结构** | 统一蓝色顶部 + 白色圆角内容区过渡，自定义导航栏 |

### 交互说明

- 登录成功后 `wx.switchTab` 跳转首页
- 已有 token 的用户 `onLoad` 时自动跳转，不停留在此页
- Token 过期（401）时从任意页面自动重定向回此页


### 交互说明

- 算式居中大字显示，数字 ≥ 36pt，运算符间距宽松
- 数字键盘占屏幕下半部分，每个按键 ≥ 60×60pt，间距 12pt
- 点击"确定"后：
  - **答对**：算式区域闪绿光 + ✅ 图标 + "叮"音效 → 0.8秒后自动进入下一题
  - **答错**：算式区域闪红光 + ❌ 图标 → 显示正确答案 1.5秒 → 自动进入下一题
- 进度条实时更新
- 左上角返回按钮：点击弹窗确认"退出将不保存本次进度，确定退出吗？"
- 计时器右上角，显示但不施加压力（无倒计时）
